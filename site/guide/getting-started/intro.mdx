# Getting started

We'll start with a basic configuration for a down-to-basics project with bud. **We should be done in about 5 minutes**.

## Using the CLI

The **bud** command is your primary means of initiating builds and carrying out other tasks.

For a better sense of what all is possible, try invoking **bud** with the `--help` flag:

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud --help

  </TabItem>
  <TabItem value="npm">

    npm run bud --help

  </TabItem>
</Tabs>
