| Utility | Description                      |
| :------ | :------------------------------- |
| [bud.alias](bud.alias) | Register shorthand for resolving modules |
| [bud.assets](bud.assets) | Include static assets in your compilation even if they aren't referenced in scripts or stylesheets |
| [bud.define](bud.define) | Replace variables in your application code and templates at compile time |
| [bud.dev](bud.dev) | Configure the development server |
| [bud.externals] | Specify a non-standard resolution strategy for modules with a matching name |
| [bud.get](bud.get) | Multi-compiler management utility allowing reference of child compilers | 
| [bud.hash](bud.hash) | Enable or disable filename hashing of built assets |
| [bud.hooks](bud.hooks/index) | Modify internal values with callbacks |
