# bud.define

[bud.define](/docs/bud.define) replaces variables in your application code and templates with specified values. 

## Signature

```ts title='bud.define.d.ts'
Bud.Define(
  values: Webpack.DefinePlugin["definitions"]
) => Bud
```

## Usage

Call **bud.define** and pass your definitions.

```ts title='bud.config.js'
bud.define({
  APP_NAME: 'My Application',
})
```

Any instances of `APP_NAME` in your application code will be replaced with `My Application`.

```ts title='src/app.js'
console.log(APP_NAME) 
// => console.log('My Application');
```

Replacements will also be made when compiling [html templates](/docs/bud.template).

```html title='public/index.html'
<html>
  <title>%APP_NAME%</title>
  <!-- ... -->
</html>
```
