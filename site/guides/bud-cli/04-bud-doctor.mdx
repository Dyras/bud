[bud doctor](/guides/bud-cli/bud-doctor) checks your package.json against the extensions you have installed and provides feedback if anything is missing.

<CodeBlock title="$ yarn bud doctor --help">{Help}</CodeBlock>

In the future [bud doctor](/guides/bud-cli/bud-doctor) will perform other checks to ensure projects are remain healthy.

## Success

Running [bud doctor](/guides/bud-cli/bud-doctor) in a project that meets requirements:

<CodeBlock title="$ yarn bud doctor">{Success}</CodeBlock>

## Failure

In the event of a failure, you may install the missing extensions with **bud init**.

<CodeBlock title="$ yarn bud doctor">{Fail}</CodeBlock>
